# 🇪🇸 Español

## 📖 Descripción

Este repositorio contiene los archivos **Mapdata** correspondientes a **Dofus Retro 1.48.17**.

Su objetivo es proporcionar una referencia para desarrolladores, investigadores y entusiastas interesados en comprender el funcionamiento interno del cliente de Dofus Retro.

### 🎯 Casos de uso

- 🔎 Investigación y análisis de datos.
- 📚 Aprendizaje sobre la estructura interna del cliente.
- 🛠 Desarrollo de herramientas y utilidades.
- 🎮 Estudio del funcionamiento de los mapas.
- 🗄 Preservación de recursos históricos del juego.
